FieldCheck Hub - CEO Dashboard

O que foi incluído:
- Novo menu "FieldCheck CEO" como tela inicial do Hub.
- Dashboard do fundador com agenda inteligente, prioridades da sprint, KPIs, saúde de clientes beta, bugs/riscos, backlog e funil comercial.
- Checklist diário salvo no navegador via localStorage.
- Layout responsivo e visual premium integrado ao FieldCheck Hub.

Como publicar:
1. Substitua os arquivos do portal por este pacote.
2. Rode localmente:
   npm install
   npm run dev
3. Se estiver tudo certo:
   git add .
   git commit -m "Adiciona FieldCheck CEO Dashboard"
   git push origin main

Observação:
O build não foi validado neste ambiente porque as dependências opcionais nativas do Vite/Rolldown não estavam completas no node_modules enviado. No seu computador, após npm install, o build deve rodar normalmente.
